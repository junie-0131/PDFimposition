# AGENTS.md

このプロジェクトは、商業印刷向けのPDF面付け・印刷支援アプリです。

Codexは作業前に、必ず以下の商業印刷ナレッジを参照してください。

- docs/domain/README.md
- docs/domain/日本の印刷面付けと製本技術の体系的解説.md
- docs/domain/印刷における面付けの基礎知識と技法管理.md
- docs/domain/綴じ方向と面付けの設計指針.md
- docs/domain/中綴じ冊子のページ数調整術.md
- docs/domain/中綴じ製本におけるクリープ処理の仕組みと計算公式.md
- docs/domain/商業印刷における面付と製本の基礎知識まとめ.md
- docs/domain/sources.md

## 重要な前提

- 「仕上がりサイズ」と「用紙サイズ」は別物として扱う。
- 「印刷する紙のサイズ」と「完成物のサイズ」は別物として扱う。
- 塗り足し、トンボ、仕上がり線、安全マージンを区別する。
- 面付けでは、ページ順、折り、綴じ方向、表裏の対応を必ず考慮する。
- 不明な仕様は勝手に決めず、ユーザーに確認する。
- 商業印刷の一般常識に反する処理を実装する場合は、理由をコメントに残す。

## 実装時の注意

- UI文言は、印刷業務に慣れていない人にも分かる日本語にする。
- 専門用語を使う場合は、必要に応じて補足説明を入れる。
- PDF処理では、mm単位とpt単位の変換ミスに注意する。
- 入稿用PDFを扱う場合、見た目だけでなく印刷事故を防ぐ設計を優先する。
